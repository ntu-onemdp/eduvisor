# eduvisor

Eduvisor integration for OneMDP

## Running the app

To run the app in development mode (with auto-reload), use

```sh
uvicorn main:app --reload
```

To run the app in production, use

```sh
uvicorn main:app
# alternatively, you can use the following command:
python main.py
```
